Dance Studio Booking System

A robust backend system for managing user authentication, class scheduling, membership cards, and attendance. Built with Java 21 and Spring Boot 3, it serves as a central REST API for frontend and mobile applications.

Features

  • Admin: Manage classes, users, and generate reports.
  • Instructor: Manage schedules, mark attendance, handle payments.
  • Client: Book classes, track membership cards, receive notifications.

Tech Stack

  • Java 21
  • Spring Boot 3
  • PostgreSQL
  • JWT Authentication
  • Flyway for Database Migrations
  • Swagger-UI for API Documentation
